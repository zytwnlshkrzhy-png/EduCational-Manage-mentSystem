# 📘 سیستم مدیریت شهریه مجتمع آموزشی

## 📌 معرفی پروژه
این پروژه به منظور **تحلیل و طراحی** یک سیستم نرم‌افزاری برای مدیریت شهریه دانش‌آموزان یک مجتمع آموزشی طراحی شده است.  
این پروژه به عنوان بخشی از درس **"تحلیل و طراحی سیستم‌ها و اصول طراحی نرم‌افزار"** تهیه شده است.

---

## 🧩 زیرسیستم‌های اصلی
1. **زیرسیستم مدیریت دانش‌آموز**  
   ثبت، ویرایش، حذف و جستجوی اطلاعات دانش‌آموزان

2. **زیرسیستم پرداخت**  
   پرداخت نقدی، پرداخت اقساطی، نمایش تاریخچه پرداخت، امکان تعریف انواع روش‌های پرداخت

3. **زیرسیستم مدیریت پرونده مالی دانش‌آموز**  
   مشاهده صورتحساب، ثبت تراکنش‌ها، محاسبه مانده بدهی و اعتبار
